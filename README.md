### Easy Setup

**Add Repository**
```shell
git clone --depth 1 https://github.com/coopersmall/nvim.git\
~/.config/
```
**Install Packer**/n
[Go here](https://github.com/wbthomason/packer.nvim)

**Install Pluging**
```
:PackerSync
```
